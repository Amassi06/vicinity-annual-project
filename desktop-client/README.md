# desktop-client Vicinity

Squelette **JavaFX 21 + Gradle Kotlin DSL** :

## Prérequis

- JDK **21**
- Gradle **8+** (`gradle run` jusqu’à ce qu’un `./gradlew` soit généré)

## Commandes

```bash
gradle run    # après résolution Maven Central du plugin JavaFX / OpenJFX
gradle build  # compilation seulement
```

Fenêtre minimale `VicinityApp` servant de point de départ pour le client admin offline-first.
